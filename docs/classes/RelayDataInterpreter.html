evaluate

  • evaluate(stateData): RelayDataInterpreter

  • Evaluate the current relay states and set the bitStates accordingly.

    The determined values are used to set the switching parameters. Therefore, the input values should be as up-to-date as possible. Otherwise, a change on one relay will reset all other relays based on the stateData.

    Important This method should be called before using any of the set operations (setOn, setOff, setAuto).

    Parameters

    • stateData: GetStateData

      The most recent GetStateData instance.
